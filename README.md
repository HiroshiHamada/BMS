# 『社会科学のためのベイズ統計モデリング』サポートサイト

このリポジトリでは『社会科学のためのベイズ統計モデリング』（朝倉書店）の文中で解説した分析や数値計算を再現するための

1. データファイル
2. Rコード
3. Stanコード

を公開しています.

## 各章のディレクトリ構造

- 一番上の階層
  - Rコードとデータファイル（csvファイルもしくはtxtファイル）が置いてあります．
- modelフォルダ
  - stanコードが置いてあります．

## 実行時のディレクトリについて

各章のディレクトリに移動してからRコードを実行してください．

異なる章のコードを実行する際には，ホームディレクトリを変更してください．
